# 1. Cálculo de notas

Se ha realizado una evaluación con alternativas para los estudiantes, y se necesita un programa que:

- Reciba el nombre del archivo con las respuestas del estudiante. Estas deben estar con el formato del archivo de ejemplo `respuestas_estudiante.txt`.
- Lea el archivo `respuestas.txt` para adquirir la respuesta correcta para cada pregunta, junto con su puntaje.
- Entregue un archivo con las respuestas del estudiante indicando si son correctas o no, el puntaje obtenido y la nota de 1.0 a 70. (ver archivo `ejemplo_salida.txt`).
